# AnonScan

A powerful server management application that runs on your system. Simply provide your server address and credentials, and AnonScan modifies DNS settings, then generates a complete management URL for full server control.

## What It Does

AnonScan is a server management tool that allows you to:

- Connect to your server using address and password
- Automatically modify DNS configurations
- Generate a complete management interface URL
- Access full server control remotely

## Key Features

- **Easy Server Connection**: Simple input of server credentials
- **Automatic DNS Management**: Modifies DNS settings automatically
- **Unified Management URL**: Single URL for complete server control
- **Remote Access**: Manage your server from anywhere
